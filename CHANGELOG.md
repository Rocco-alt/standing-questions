# Changelog

Notable changes to the pattern, skill, schemas, and metrics. Semantic versioning applies to the set: patch = wording/docs, minor = compatible additions (new optional fields, new questions in the seed set), major = breaking changes to the schemas or the protocol.

## 1.0.0 — 2026-06-07

Initial public release: the pattern write-up (README), the field story (BLOG), two JSON Schemas (draft 2020-12), the generic 12-question seed set, the 7-session sample log, the installable Claude Code skill, and the two stdlib-only metrics with tests.
